# AutoCorrect
This is an auto correct problem with edit distance

## How to run this code.

### compile: 
`mvn clean install`

### run: 
`java -cp target/auto-correct-1.0-SNAPSHOT.jar com.oracle.main.AutoCorrect src/main/resources/words.txt`
